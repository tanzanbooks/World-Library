- # 📚 知識循環ダッシュボード

World Library Project のために開発している
Python + Streamlit ダッシュボードです。

## 目的

金融データ分析の考え方を

- 図書館
- 古本屋
- 読書会
- 一箱古本市

へ応用し、

知識循環を可視化することを目的としています。

## 現在の機能

- 売上分析
- 粗利益分析
- 棚主別分析
- ジャンル分析
- 在庫分析
- イベント分析
- SNS効果分析

## 将来予定

- 世界の図書館マップ
- 図書館貸出分析
- 知識循環指数
- AIによる需要予測

## 1. インストール

WindowsのコマンドプロンプトまたはPowerShellで、ファイルを保存したフォルダへ移動して実行します。

```powershell
python -m pip install -r requirements.txt
```

## 2. 起動

```powershell
streamlit run app.py
```

ブラウザが開いたら、左側から `sample_data.csv` または自分のExcelファイルを読み込みます。

## 3. 必要な列

- 商品ID
- 書名
- 著者
- ジャンル
- 棚主
- 仕入日
- 仕入価格
- 出品日
- 販売日
- 販売価格
- 販売チャネル
- イベント名
- イベント日
- SNS紹介日

販売前の本は「販売日」を空欄にします。

## 4. 分析できる内容

- 登録冊数、販売冊数、在庫冊数
- 売上、粗利益、販売率
- 月別売上推移
- 棚主別・ジャンル別成績
- 長期在庫
- イベント後30日以内の販売率
- SNS紹介後30日以内の販売率
